Q.1 If  $\left(x - \frac{1}{2}\right)^2 - \left(x - \frac{3}{2}\right)^2 = x + 2$ , then the value of x is :

(a) 2

(b) 8

(c) 4

(d) 6

Ans. (c)

$$\left(x - \frac{1}{2}\right)^2 - \left(x - \frac{3}{2}\right)^2 = x + 2$$
Using
$$a^2 - b^2 = (a + b)(a - b)$$

$$\left(x - \frac{1}{2} + x - \frac{3}{2}\right)\left(x - \frac{1}{2} - x + \frac{3}{2}\right) = x + 2$$

$$(2x - 2) = x + 2$$

$$2x - x = 2 + 2$$

$$x = 4$$
